# WeatherPage

中国の無料公開天気APIを利用して、ダミーページを作成しました。

グラフはChart.jsを利用しています。モーダルとレスポンスはBootstrapを利用しています。

テーブルの生成とデータの加工はjQueryのdataTableで実装しました。
